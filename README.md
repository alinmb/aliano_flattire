A 100% standalone FiveM script that allows players to slash vehicle tires using melee weapons. Fully synchronized and optimized for RP servers.

## Features

- **Smart Detection**: Automatically detects the closest tire when holding a melee weapon
- **Visual Feedback**: 3D text display key prompt
- **Realistic Animation**: 3-second slashing animation with progress bar
- **Full Synchronization**: Tire damage is synchronized for all players
- **Anti-Spam**: Prevents slashing already flat tires
- **Fully Configurable**: All weapons, animations, and messages in config.lua
- **Smart Sync**: Different sync methods for empty vs occupied vehicles
- **Interruption System**: Action cancels if player moves away, vehicle moves, or gets hit
